Executando o programa pelo terminal
Baixe e coloque os códigos-fonte (Aresta.java, Grafo.java, Tela.java, Vertice.java)
em uma mesma pasta.
E no terminal de comandos, compile o programa:
javac Tela.java
E execute o programa:
java -cp . Tela

Gerando executável .jar a partir do NetBeans IDE:
Crie um Novo Projeto "Tela" e gere o arquivo .jar

Ou baixe o arquivo tela.java.jar e execute
